# Simple UNO-like Card Game (Java)

A simple Java-based card game inspired by UNO.  
Implements core mechanics: matching by color or number, basic action cards (Skip, Reverse, Draw Two), and turn-based gameplay. Designed for learning, experimentation, and small hobby projects.

## Features
- Match by color or number.
- Action cards: Skip, Reverse, Draw Two (basic behavior).
- Turn-based play with a basic CPU player (or multiple CPU players).
- Simple CLI (console) interface for gameplay.
- Clean, easy-to-extend code structure.

## Requirements
- Java 11+ (works with Java 8+ in most places, but Java 11 recommended)
